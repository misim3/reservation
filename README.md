# reservation

## 1. 주요 스택
- jdk : 19
- Spring JDBC
- Tomcat
- MySQL


## 2. 패키지 구조

    main
    ├── java
          ├── Todo - 할일 목록과 관련된 클래스
          └── main - 메인 화면와 관련된 클래스
    └── webapp
          ├── css
          ├── images
          ├── js
              ├── main.js - ajax 비동기 통신
              └── todoForm.js - 할일 등록 값 유효성 확인
          └── jsp - 화면 출력
